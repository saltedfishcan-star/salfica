# ATD 学习文档（新手版，纯前端）

## 1. 这个项目做什么
你可以用它来：

1. 添加图片（本地文件或图片直链 URL）
2. 配置榜单等级和颜色
3. 拖拽图片进行排序
4. 导出榜单 PNG

## 2. 先看目录
```text
src/
  lib/board.ts           # 数据模型与本地存储
  views/SetupView.vue    # 配置页
  views/BoardView.vue    # 榜单页
  router/index.ts        # 路由
```

## 3. 怎么运行
在项目根目录执行：

```bash
npm install
npm run dev
```

## 4. 推荐阅读顺序
1. `src/lib/board.ts`：先理解数据结构与持久化。
2. `src/views/SetupView.vue`：看图片导入、错误提示、进入榜单流程。
3. `src/views/BoardView.vue`：看拖拽和导出逻辑。

## 5. 你要知道的限制
- 只支持图片直链 URL，不支持网页链接自动提图。
- 部分外链图片可能被源站限制加载，会显示失败态。
