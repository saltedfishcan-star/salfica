# ATD 项目学习文档（纯前端全景）

## 1. 项目定位
ATD 是一个纯前端图片榜单工具，核心能力：

- 导入图片（本地文件 + 图片直链 URL）
- 配置等级并拖拽排序
- 导出榜单区域 PNG

当前版本不含后端，因此没有接口调用与服务端代理流程。

## 2. 技术组成
- Vue 3 + `<script setup lang="ts">`
- Vue Router
- vuedraggable（列表间拖拽）
- html2canvas（区域截图导出）
- Vite（开发与构建）

## 3. 关键模块

### 3.1 `src/lib/board.ts`
负责业务数据模型和本地持久化：

- 类型：`BoardConfig`、`BoardState`、`ImageItem`、`TierConfig`
- 常量约束：等级数量、上传上限、大小限制
- 读写：`load/save/reset`，并带内存兜底

这是项目稳定性的核心模块。

### 3.2 `src/views/SetupView.vue`
负责配置和导入：

- 等级增删、排序、颜色设置
- 本地文件导入
- 文本/拖拽 URL 提取
- 直链图片可访问性检测与尺寸读取
- 失败态反馈（不可访问、非直链）

### 3.3 `src/views/BoardView.vue`
负责榜单操作：

- 各等级区与待排序池拖拽互通
- 拖拽状态实时持久化
- 仅导出 `rank-table` 区域为 PNG

## 4. 运行与构建
项目根目录执行：

```bash
npm install
npm run dev
```

构建：

```bash
npm run build
```

## 5. 部署注意
`vite.config.ts` 已设置：

```ts
base: '/salfica/'
```

用于 GitHub Pages 项目站点（`saltedfishcan-star/salfica`）。

## 6. 推荐阅读顺序
1. `src/lib/board.ts`
2. `src/views/SetupView.vue`
3. `src/views/BoardView.vue`
4. `src/router/index.ts`
